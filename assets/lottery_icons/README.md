## IMPORTANT: Trademark Notice

Official lottery logos are trademarked and copyrighted by their respective lottery operators. 

**For production use, you must:**
1. Obtain permission from the lottery operator
2. Use only for reference/educational purposes
3. Create your own derivative icons that don't infringe trademarks
4. Use generic lottery-themed icons (balls, tickets, etc.)

## Alternative: Generic Icons

Instead of official logos, you can use generic lottery-themed icons:
- Lottery ball with number

## Installation Instructions:

1. Obtain appropriate icons (official with permission OR generic)
2. Resize to 64x64 pixels
3. Save as PNG with transparency
