   - Colors: Blue and white stripes with cross
   - Size: 64x64 pixels recommended
   - Format: PNG with transparency

2. **european_union.png** - Flag of European Union (for EuroJackpot)

   - Size: 64x64 pixels recommended

   - Size: 64x64 pixels recommended
   - Format: PNG with transparency

   - Red and yellow horizontal stripes
   - Size: 64x64 pixels recommended
   - Format: PNG with transparency

5. **italy.png** - Flag of Italy (for SuperEnalotto)
   - Green, white, red vertical stripes
   - Size: 64x64 pixels recommended
   - Format: PNG with transparency

6. **france.png** - Flag of France (for Loto France)

   - Size: 64x64 pixels recommended

7. **germany.png** - Flag of Germany (for Lotto 6aus49)

8. **austria.png** - Flag of Austria (for Austrian Lotto)

   - Size: 64x64 pixels recommended

## Installation Instructions:

1. Download country flags from a free source (see ASSETS_README.md)
2. Resize to 64x64 pixels
3. Save as PNG with transparency
4. Place in this directory with the exact filenames listed above
